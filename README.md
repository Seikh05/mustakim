# Developer Portfolio & Blog — Seikh Souvagya Mustakim

A clean, minimalist, and responsive developer portfolio and blog. Built using semantic HTML5, vanilla CSS3, and JavaScript, with dynamically loaded sections that pull data from a single centralized data file.

## 🚀 How to Run
Open **`index.html`** directly in any modern web browser. No compilation or local server is required.

## 📁 Project Structure
* **`index.html`** — Homepage displaying your profile, skills, experience timeline, projects, achievements, interests, and extracurriculars.
* **`blog.html`** — Archive list of your personal notes/blog logs.
* **`post.html`** — Clean layout to view individual markdown blog logs.
* **`portfolio-data.js`** — Centralized database containing all profile text, links, experiences, projects, and blog content. Modify this file to update your portfolio.
* **`assets/site.js`** — The dynamic rendering script engine and dark/light mode toggle controller.
* **`assets/style.css`** — Design system featuring custom styling variables, responsive breakpoints, and vibrant color accent styling.
* **`assets/MyCV.pdf`** — Your downloadable CV resume.

## 🛠️ Customization
To change any information on the site, simply open **`portfolio-data.js`** and edit the parameters inside the global `window.portfolioData` object.
